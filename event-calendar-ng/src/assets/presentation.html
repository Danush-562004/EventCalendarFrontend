02 / 05
◆
EVENTCAL
Unique Features
What makes this platform stand out
💸 Smart Refund System
  • Admin cancels event → auto 100% refund triggered
  • Only applies if event hasn't started yet
  • Payments marked Refunded, tickets Cancelled atomically
  • Refund email sent to every affected user
⏱ 5-Minute Payment Window
  • Ticket reserved for exactly 5 minutes after booking
  • Live countdown timer on the ticket card
  • Backend enforces the deadline — cannot be bypassed
  • Expired tickets blocked from payment server-side
🔔 Real-Time Email & In-App Notifications
  • Booking confirmation email with full ticket details
  • Scheduled reminder emails via background service
  • In-app bell with unread count, polls every 30s
  • Refund alert delivered both in-app and by email
🎟 Ticket & Capacity Management
  • Max 10 tickets per user per event (enforced backend)
  • Venue capacity enforced at event creation
  • Available seats update reactively after each booking
  • No-refund policy acknowledgement before payment
